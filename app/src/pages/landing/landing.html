Adobe Summit 2019 Session 202

This mobile app is used in conjunction with Adobe Summit S202.

  1. This app consumes content from Adobe Experience Manager 6.5 using Content Services.

  2. Summit Speakers Menu consumes Content Fragments exposed via Content Services API Pages.

  3. Summit FAQ Menu consumes Content Fragments exposed via the Content Fragment HTTP API.

  4. Products Menu uses Sling Model Exporters to expose existing We-Retail products from AEM.
